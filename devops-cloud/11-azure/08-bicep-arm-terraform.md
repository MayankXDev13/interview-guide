## Bicep vs ARM vs Terraform

### Question

How do you create Azure resources: Bicep, ARM or Terraform?

### Short explanation of the question

This evaluates Infrastructure as Code knowledge.

### Answer

I primarily use Terraform for multi-cloud consistency. Bicep is
preferred for Azure-native deployments, while ARM templates are the
underlying deployment format.

### Typical interview answer

> I mainly use Terraform. For Azure-native projects Bicep is also an
> excellent choice because it is simpler than ARM templates.

### Key takeaways

-   Terraform: multi-cloud.
-   Bicep: Azure native.
