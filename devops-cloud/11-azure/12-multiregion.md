## Multi-Region Data Redundancy and Performance

### Question

How do you ensure data redundancy and performance in multi-region
deployment?

### Short explanation

Tests HA and DR design.

### Answer

Deploy across regions, use Traffic Manager/Front Door, geo-redundant
storage, SQL geo-replication and regional failover.

### Typical interview answer

> I use active-active or active-passive architecture with
> geo-replication and global load balancing.

### Key takeaways

-   Geo replication.
-   Global load balancing.
