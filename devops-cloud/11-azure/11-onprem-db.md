## Connect Azure VMs to an On-Premises Database

### Question

Explain how to connect Azure VMs to a database that is on-premises.

### Short explanation

Tests hybrid networking.

### Answer

Use Site-to-Site VPN or ExpressRoute with private routing.

### Typical interview answer

> I connect Azure VNets to on-premises using VPN or ExpressRoute and
> secure traffic with NSGs and firewalls.

### Key takeaways

-   VPN.
-   ExpressRoute.
