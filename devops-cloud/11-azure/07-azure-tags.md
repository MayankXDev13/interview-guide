## Azure Resource Tags

### Question

How do you use tags on Azure resources? Do you enforce them?

### Short explanation of the question

Tags help organize, manage and allocate costs.

### Answer

I use tags such as Environment, Owner, CostCenter and Project. In
production I enforce mandatory tags using Azure Policy.

### Typical interview answer

> We use Azure Policy to require tags during resource creation for
> governance and cost reporting.

### Key takeaways

-   Tags improve governance.
-   Azure Policy enforces compliance.
