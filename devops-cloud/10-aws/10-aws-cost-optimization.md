## AWS Cost Optimization Activity

### Question

Explain a cost optimization activity that you performed in the current
organization.

### Short explanation of the question

This question evaluates your experience reducing AWS costs without
affecting application performance.

### Answer

Typical optimization activities include: - Right-sizing EC2 instances. -
Deleting unused EBS volumes and snapshots. - Purchasing Savings Plans or
Reserved Instances. - Enabling S3 lifecycle policies. - Stopping idle
development resources. - Using Auto Scaling.

### Detailed explanation

Example: - Reduced oversized EC2 instances from `m5.large` to
`t3.medium`. - Configured Auto Scaling. - Deleted unattached EBS
volumes. - Moved S3 objects to Glacier.

Result: - Lower monthly infrastructure cost without impacting
availability.

### Typical interview answer

> I reviewed AWS Cost Explorer, right-sized EC2 instances, removed
> unused storage, enabled lifecycle policies, and configured Auto
> Scaling, which reduced costs while maintaining application
> performance.

### Key takeaways

-   Monitor costs regularly.
-   Right-size resources.
-   Remove unused infrastructure.
