## Day-to-Day Database Activities

### Question

Explain your day-to-day activities related to databases.

### Short explanation of the question

This question checks operational database experience.

### Answer

Typical activities include monitoring database health, backups,
performance tuning, security, storage management, and troubleshooting.

### Detailed explanation

Daily tasks: - Monitor CPU, memory, storage and connections. - Review
slow queries. - Verify backups. - Check replication health. - Rotate
credentials. - Monitor CloudWatch alarms.

### Typical interview answer

> My daily work includes monitoring RDS health, checking storage and
> connections, reviewing slow queries, validating backups, managing
> access, and troubleshooting performance issues.

### Key takeaways

-   Monitor.
-   Backup.
-   Optimize.
-   Secure.
