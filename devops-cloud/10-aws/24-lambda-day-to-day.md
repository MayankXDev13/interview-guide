## Day-to-Day Activities with AWS Lambda

### Question

Have you worked with Lambda in AWS? Explain your activities.

### Short explanation of the question

Interviewers want practical Lambda experience rather than theory.

### Answer

Yes. I have used Lambda for event-driven automation and backend
processing.

### Detailed explanation

Typical activities: - Develop Lambda functions. - Trigger from API
Gateway, S3, EventBridge or SQS. - Monitor CloudWatch Logs and
Metrics. - Configure IAM roles. - Optimize timeout and memory. - Deploy
using CI/CD.

### Typical interview answer

> I develop and deploy Lambda functions, integrate them with AWS
> services, monitor them through CloudWatch, troubleshoot failures, and
> optimize memory and execution time.

### Key takeaways

-   Event-driven.
-   Serverless.
-   Monitor with CloudWatch.
