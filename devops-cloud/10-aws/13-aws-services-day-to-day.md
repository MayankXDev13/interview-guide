## AWS Services Used in Day-to-Day Activities

### Question

Which AWS services do you use in your day-to-day life?

### Short explanation of the question

This question evaluates your hands-on AWS experience.

### Answer

Typical daily services include EC2, VPC, IAM, ALB, Auto Scaling,
EKS/ECS, Lambda, S3, RDS, CloudWatch, CloudTrail, Route 53, Secrets
Manager, ECR, and Systems Manager.

### Detailed explanation

Common daily tasks: - Deploy applications on EKS/EC2. - Monitor with
CloudWatch. - Store objects in S3. - Manage IAM roles and policies. -
Build CI/CD with ECR. - Troubleshoot using CloudTrail.

### Typical interview answer

> I regularly use EC2, EKS, IAM, VPC, ALB, Auto Scaling, RDS, S3,
> CloudWatch, CloudTrail, Route53 and Secrets Manager for deployment,
> monitoring and troubleshooting.

### Key takeaways

-   Mention only services you've used.
-   Explain practical usage, not definitions.
